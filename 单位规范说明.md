## 📐 单位分类汇总

### 长度单位
- **m** (米) - 用于距离、深度、长度、宽度、厚度、直径等

### 面积单位
- **m²** (平方米) - 用于采空区面积、积水面积、火区面积、悬顶面积、塌陷面积等

### 体积单位
- **m³** (立方米) - 用于积水量、治理量等

### 浓度单位
- **%** (百分比) - 用于气体浓度

### 温度单位
- **℃** (摄氏度) - 用于火区温度

## 📝 使用说明

### 1. 数据录入时

**推荐做法**:
```csv
# 使用默认单位，单位字段可留空
goaf_area,area_unit
1000,

# 或明确填写单位
goaf_area,area_unit
1000,m²
```

**特殊情况**:
```csv
# 如果使用非默认单位，必须明确填写
goaf_area,area_unit
1,km²

# 或
goaf_area,area_unit
10000,m²
```

### 2. 数据处理时

- 如果单位字段为空，默认使用Schema定义的默认单位
- 如果单位字段有值，以实际填写的单位为准
- 建议统一使用默认单位，便于数据比对和分析

### 3. 单位转换

如需单位转换，建议：

**面积转换**:
- 1 km² = 1,000,000 m²
- 1 公顷 = 10,000 m²

**体积转换**:
- 1 万m³ = 10,000 m³

**浓度转换**:
- 1% = 10,000 ppm

## ✅ 数据验证规则

### 必须遵守的规则

1. **数值字段**: 必须是数字，不能包含单位符号
   ```csv
   ✅ 正确: goaf_area = 1000
   ❌ 错误: goaf_area = 1000m²
   ```

2. **单位字段**: 如果填写，必须是有效的单位符号
   ```csv
   ✅ 正确: area_unit = m²
   ✅ 正确: area_unit = km²
   ❌ 错误: area_unit = 平方米
   ```

3. **一致性**: 同一表中相同类型的字段应使用相同单位
   ```csv
   ✅ 推荐: 所有面积都用m²
   ⚠️ 不推荐: 有的用m²，有的用km²
   ```

## 🎯 最佳实践

### 推荐做法

1. **统一使用默认单位** - 便于数据比对
2. **单位字段留空** - 系统自动使用默认单位
3. **特殊情况明确标注** - 使用非默认单位时必须填写
